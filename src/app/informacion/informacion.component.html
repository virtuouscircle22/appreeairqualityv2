In the Current Data section we can find the data collected in three different cities (each city has three stations). In Seville the stations used to collect data are: Los Principes, Bermejales and Torneo, in Sofia the stations are: Pavlovo, Druzhba and Hipodruma, while in Athens they are: Pireaus, LykovSrisi and Agia Paraskevi. The data are public and can be accessed by any person through this link

In the Monthly Data we can see the average value for each pollutant on each day of the month and country chosen (for a specific station). This data is stored in our own database, where we store the values of each pollutant for each month and country. We store the values of each pollutant every time the air quality is measured at each of the selected stations of Sofia, Athens and Seville.

In the Annual Data tab you can see the average of the values for each day of the pollutant, country (for a specific station) and year chosen. As before, this data is stored in our own database.

We can also see comparison of the values of the different pollutants in a date range and station that we have chosen. These data are in the same database as the database as the data of the two previous sections.

In the PM10 Evolution section it shows the PM10 values in a range of dates and countries that we have selected. These data are obtained through devices that are located in the secondary schols of each city. These devices have been built by us. They measure the value of the PM10 that is stored in our databases.
